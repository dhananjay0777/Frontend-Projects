# Frontend Projects

This repository contains my frontend projects built with HTML, CSS, and JavaScript. I use it to practice web development, explore new ideas, and build interactive web applications.

## Projects

### 1. Kanban To-Do List

A simple Kanban-style task manager that lets you organize your tasks into different stages.

#### Features

- Create new tasks
- Add a title and description
- Drag and drop tasks between columns
- Organize tasks into:
  - To-Do
  - In Progress
  - Done
- Automatically saves tasks using Local Storage
- Displays the number of tasks in each column

#### Built With

- HTML
- CSS
- JavaScript
- Local Storage API

## Getting Started
